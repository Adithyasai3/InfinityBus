# InfinityBus

A backend system for online bus reservation and management built using Node.js, Express, TypeScript, Prisma, and PostgreSQL.

## Features

### Authentication

* User Registration
* User Login
* Password Hashing with bcrypt
* JWT Authentication
* Protected Routes

### Authorization

* Role-Based Access Control
* ADMIN
* OPERATOR
* PASSENGER

### Bus Management (In Progress)

* Create Bus
* View Buses
* Manage Buses

## Tech Stack

* Node.js
* Express.js
* TypeScript
* PostgreSQL
* Prisma ORM
* JWT
* bcrypt
